static uint8_t irq_nested_disable ( )
inlinestatic

Disable IRQs (nested).

Disable IRQs when unsure if IRQs were enabled at all. This function together with irq_nested_enable can be used in situations when interrupts shouldn't be activated if they were not activated before calling this function.
